# 🌞 Smart Solar Tracking Dashboard

A modern Python-based desktop dashboard for monitoring solar tracking system data in real-time.  
Built using **Tkinter, SQLite, Pandas, and Matplotlib**.

---

## 🚀 Features

- 📊 Real-time data visualization (Voltage & Temperature graphs)
- 📋 Scrollable data table with latest sensor readings
- 🖱️ Full-page scrolling with mouse wheel support
- ⚡ Live dashboard auto-refresh (every 2 seconds)
- 🧠 Clean and responsive UI design
- 💾 SQLite database integration

---

## 🛠️ Tech Stack

- Python 3
- Tkinter (GUI)
- SQLite (Database)
- Pandas (Data handling)
- Matplotlib (Graph plotting)
